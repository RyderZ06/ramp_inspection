Структура проекта
- `src/` — исходный код, разделённый на функции, библиотеки и UI.
- `data/` — хранение баз данных вносимых через графический интерфейс программы, если нет ссылок на утвержденные базы данных в компании.
- `venv_ramp/` — виртуальное окружение.
- `info_list/` — хранение памяток по работе с программой.
- `requirements.txt` — зависимости.
- `README.txt` — описание структуры проекта.
- `README.md` — описание проекта.
