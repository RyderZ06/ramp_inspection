Создайте файл .gitignore командой: cmd: "echo.>.gitignore"

Загрузить расширение для удобной работы с кодом в Jupyter Lab: jupyterlab-indent-guides

Работа с Git & GitHub:

1. Добавьте обновленный файлы в индекс: git add -A
2. Сделайте коммит: git commit -m "Initial commit"
------------------------------------------------
Удалите папку venv_ramp из индекса Git (но не из файловой системы)
Выполните команду: git rm -r --cached venv_ramp
* Опция --cached удалит папку из индекса (то есть из отслеживаемых Git файлов), но оставит её на диске.
------------------------------------------------
3. Сделайте коммит с удалением виртуального окружения из репозитория: git commit -m "Remove venv_ramp from repository"
4. Отправьте изменения в удалённый репозиторий: git push origin master
